# ChadAI

A simple AI chat simulator built with HTML, CSS, and JavaScript. This project demonstrates a basic chat interface where users can send messages and receive simulated AI responses.

## Features

- Interactive chat interface
- Simulated AI typing indicator
- Random AI responses from a predefined set
- Responsive design
- Session-based message handling

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/sfezekile/AI-bot.git
   ```

2. Navigate to the project directory:
   ```
   cd AI-bot
   ```

3. Open `main.html` in your web browser to start the application.

## Usage

1. Open `main.html` in a web browser.
2. Enter your message in the input field and submit.
3. The app will redirect to `chat.html` and simulate an AI response.
4. The AI will display a typing indicator before responding with a random message.

## Project Structure

- `main.html`: The main entry point with the input form.
- `app.js`: JavaScript logic for handling chat interactions.
- `style.css`: Styles for the main page.
